# FLUTE

Fast Light Ultimate Text Editor.

Uses FLTK: `sudo apt-get install libfltk1.3-dev`

You need some other libraries too, libpng, libjpeg, libxinerama

Default Keyboard Shortcuts:
*	C-b		Previous Buffer
*	C-o		Open File
*	C-q		Quit
*	C-s		Save File
*	C-w		Close File
